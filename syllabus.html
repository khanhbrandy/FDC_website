PREREQUISITES

Applicants are required to complete the following modules (or equivalencies) before applying for the Data Science in Economics and Business program:

  • Advanced Mathematics (details)
  • Statistics and Probability (details)

Courses listed HERE are equivalent to the two courses above and can be taken as prerequisites.

Database Management System

This course introduces students to important aspects of databases, database manipulation and database management systems. The course is based on the relational model. Learners will explore the tools and methods for database design and manipulation as well as the programming of database applications. Part of the practical experience they will gain will be acquired using a modern relational database management system. Students will also gain programming experience using SQL, and using a high level programming language to write applications that access the database.

Programming for Data Analysis and Scientific Computing

This course discusses how Python can be utilized in scientific computing and how this programming language is used: related scientific libraries, reproducibility, and the broader ecosystem of science in Python. Students will learn the foundations of programming in Python for data science, and how to conduct data analysis. The courses will also introduces the main packages for numerical computing (NumPy, SciPy), Pandas for processing tabular data, Matplotlib for data visualization as well as Sklearn for machine learning.

Advanced Analytics in Economics and Business

During this course, students will be equipped with data processing, computing and mining capabilities in order to generate actionable insights and forecasts, helping organizations fully utilize their data and determine the best course of action for every real-world problem within a range of business contexts, thereby making decisions in a more timely and efficient manner. The subject content is closely tied to the practice of Data Analytics and Business Intelligence.
